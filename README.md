# 🎯 Overview
A self-contained, responsive web application for engineering students to master **Digital Integrated Circuits** concepts. The platform hosts **322 peer-reviewed questions** covering:
- DRAM architectures (Open, Folded, Bilevel digitlines)
- Sense amplifiers & isolation techniques
- Domino logic & dynamic circuit design
- Memory hierarchies (SRAM, DRAM, Flash)
- Device modeling (CMOS, BJT, FinFET)
- Timing analysis & clock distribution

**Live Demo:** *[https://obidombieonimuonyamfo.github.io/Digital_IC_Q-A/]*

## How It Works

1. **Select an answer** – Click any option (A, B, C, or D)
2. **Instant feedback** – Card turns green (correct) or red (incorrect)
3. **Answer locks** – Cannot change after selection (exam discipline)
4. **View explanation** – Click the 💡 bulb for AI-powered explanation
5. **Track progress** – Monitor stats and progress bar
6. **Retake wrong answers** – Click "Retake Wrong" to practice only mistakes
